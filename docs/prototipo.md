# Prototipo de Solución (Wireframes)

## Pantallas sugeridas
1. **Inicio / KPIs**
   - Tarjetas con Total Sales, Avg Ticket, Units Sold, Hubs.
2. **Top 5 Productos**
   - Tabla con ranking y total vendido.
3. **Insights**
   - Lista con 3 insights accionables en inglés.
4. **Galería de Dashboards**
   - Capturas de Power BI con modal.

## Herramientas sugeridas
- Figma
- Canva
- Draw.io

> El prototipo debe mostrar navegación básica entre secciones.
